# eda-leda-Graphics-projects
Este projeto tem como foco estudar a identificação de Componentes Fortemente Conectados (SCC – Strongly Connected Components) em grafos direcionados. Para isso, serão estudados e implementados dois algoritmos clássicos baseados em Busca em Profundidade (DFS – Depth-First Search): o algoritmo de Kosaraju e o algoritmo de Tarjan
